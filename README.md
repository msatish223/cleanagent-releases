## What It Does

CleanAgent scans your computer, finds what's slowing it down, and fixes it — safely, with one click.

**🔍 Smart Scanning**
- Real-time CPU, memory, and disk monitoring with performance scoring
- Cache analysis — browser, system, pip, Homebrew
- Large file detection (100MB+) across Downloads, Documents, Desktop
- Hash-based duplicate file finder
- Old downloads audit (30+ days)
- Grouped process monitor with real CPU usage

**🧹 Safe Cleanup**
- One-click cleanup: browser caches, system logs, temp files, trash, DNS
- Selective duplicate removal — choose which copies to keep
- Old downloads deletion with checkboxes
- pip & Homebrew cache purge
- Before/After comparison showing exact disk recovered and score improvement

**🛡️ Privacy First**
- 100% local — no data leaves your machine
- No accounts, no telemetry, no analytics
- No internet connection required
- Open confirmation dialogs before every cleanup action

## FAQ

**Is it safe?**
Yes. Every cleanup shows a confirmation dialog. CleanAgent never touches system files, passwords, bookmarks, or personal documents.

**Is it really free?**
Yes. No trial limits, no feature gates, no hidden costs. All features are included.

**Does it collect my data?**
No. Everything runs locally. Zero telemetry, zero analytics, zero network calls.

**Why does macOS block it on first launch?**
The app is not yet signed with an Apple Developer certificate. Right-click → Open bypasses this once. We're working on official signing.
